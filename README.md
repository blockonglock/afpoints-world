# AF.POINTS

Minimal static site for `afpoints.world`, built with Astro and deployed to GitHub Pages.

## Local Commands

```sh
npm install
npm run dev
npm run check
npm run build
npm run preview
```

## Security Notes

- This repository is intended to be public and must not contain backend code, admin panels, API keys, or old git history.
- The old private repository should stay private because its history contains rotated secrets.
- Rotate the old MongoDB, Spotify, OpenAI, and admin credentials before making any Pages repo public.
